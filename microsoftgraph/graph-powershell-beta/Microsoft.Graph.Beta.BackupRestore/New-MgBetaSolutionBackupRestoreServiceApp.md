# New-MgBetaSolutionBackupRestoreServiceApp

## SYNOPSIS

Create a new serviceApp.

## SYNTAX

### CreateExpanded (Default)

```
New-MgBetaSolutionBackupRestoreServiceApp [-ResponseHeadersVariable <string>]
 [-AdditionalProperties <hashtable>] [-Application <IMicrosoftGraphIdentity>]
 [-EffectiveDateTime <datetime>] [-Id <string>] [-LastModifiedBy <IMicrosoftGraphIdentitySet>]
 [-LastModifiedDateTime <datetime>] [-RegistrationDateTime <datetime>] [-Status <string>] [-Break]
 [-Headers <IDictionary>] [-HttpPipelineAppend <SendAsyncStep[]>]
 [-HttpPipelinePrepend <SendAsyncStep[]>] [-Proxy <uri>] [-ProxyCredential <pscredential>]
 [-ProxyUseDefaultCredentials] [-WhatIf] [-Confirm] [<CommonParameters>]
```
