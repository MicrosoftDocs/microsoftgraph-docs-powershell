# New-MgBetaSolutionBackupRestoreDriveInclusionRule

## SYNOPSIS
Create new navigation property to driveInclusionRules for solutions

> [!NOTE]
> To view the v1.0 release of this cmdlet, view [New-MgSolutionBackupRestoreDriveInclusionRule](/powershell/module/Microsoft.Graph.BackupRestore/New-MgSolutionBackupRestoreDriveInclusionRule?view=graph-powershell-1.0)

## SYNTAX

### CreateExpanded (Default)
```
New-MgBetaSolutionBackupRestoreDriveInclusionRule [-ResponseHeadersVariable <String>]
 [-AdditionalProperties <Hashtable>] [-CreatedBy <IMicrosoftGraphIdentitySet>] [-CreatedDateTime <DateTime>]
 [-DriveExpression <String>] [-Error <IMicrosoftGraphPublicError>] [-Id <String>] [-IsAutoApplyEnabled]
 [-LastModifiedBy <IMicrosoftGraphIdentitySet>] [-LastModifiedDateTime <DateTime>] [-Status <String>]
 [-Headers <IDictionary>] [-ProgressAction <ActionPreference>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### Create
```
New-MgBetaSolutionBackupRestoreDriveInclusionRule -BodyParameter <IMicrosoftGraphDriveProtectionRule>
 [-ResponseHeadersVariable <String>] [-Headers <IDictionary>] [-ProgressAction <ActionPreference>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```
