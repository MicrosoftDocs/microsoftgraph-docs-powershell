# New-MgBetaSolutionBackupRestoreSiteProtectionUnitBulkAdditionJob

## SYNOPSIS

> [!NOTE]

```
New-MgBetaSolutionBackupRestoreSiteProtectionUnitBulkAdditionJob [-ResponseHeadersVariable <string>]
 [-AdditionalProperties <hashtable>] [-CreatedBy <IMicrosoftGraphIdentitySet>]
 [-CreatedDateTime <datetime>] [-DisplayName <string>] [-Error <IMicrosoftGraphPublicError>]
 [-Id <string>] [-LastModifiedBy <IMicrosoftGraphIdentitySet>] [-LastModifiedDateTime <datetime>]
```
