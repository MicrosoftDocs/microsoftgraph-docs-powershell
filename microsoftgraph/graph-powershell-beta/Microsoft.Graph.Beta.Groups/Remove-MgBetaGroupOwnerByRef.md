# Remove-MgBetaGroupOwnerByRef

## SYNOPSIS
Use this API to remove an owner from a Microsoft 365 group or a security group through the owners navigation property.
When owners are assigned to a group, the last owner (a user object) of the group cannot be removed.

## SYNTAX

### Delete (Default)
```
Remove-MgBetaGroupOwnerByRef -DirectoryObjectId <String> -GroupId <String> [-IfMatch <String>]
 [-ResponseHeadersVariable <String>] [-Headers <IDictionary>] [-PassThru] [-ProgressAction <ActionPreference>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

### Delete1
```
Remove-MgBetaGroupOwnerByRef -GroupId <String> -Id <String> [-IfMatch <String>]
 [-ResponseHeadersVariable <String>] [-Headers <IDictionary>] [-PassThru] [-ProgressAction <ActionPreference>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

### DeleteViaIdentity1
```
Remove-MgBetaGroupOwnerByRef -InputObject <IGroupsIdentity> -Id <String> [-IfMatch <String>]
 [-ResponseHeadersVariable <String>] [-Headers <IDictionary>] [-PassThru] [-ProgressAction <ActionPreference>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

Remove-MgBetaGroupOwnerByRef -InputObject <IGroupsIdentity> [-IfMatch <String>]
 [-ResponseHeadersVariable <String>] [-Headers <IDictionary>] [-PassThru] [-ProgressAction <ActionPreference>]

## DESCRIPTION
Use this API to remove an owner from a Microsoft 365 group or a security group through the owners navigation property.
When owners are assigned to a group, the last owner (a user object) of the group cannot be removed.
