# Confirm-MgBetaUserChatPermissionGrantMemberGroup

## SYNOPSIS
Check for membership in a specified list of group IDs, and return from that list those groups (identified by IDs) of which the specified user, group, service principal, organizational contact, device, or directory object is a member.
This function is transitive.
You can check up to a maximum of 20 groups per request.
This function supports all groups provisioned in Azure AD.
Because Microsoft 365 groups cannot contain other groups, membership in a Microsoft 365 group is always direct.

> To view the v1.0 release of this cmdlet, view [Confirm-MgUserChatPermissionGrantMemberGroup](/powershell/module/Microsoft.Graph.Users.Actions/Confirm-MgUserChatPermissionGrantMemberGroup?view=graph-powershell-v1.0)
