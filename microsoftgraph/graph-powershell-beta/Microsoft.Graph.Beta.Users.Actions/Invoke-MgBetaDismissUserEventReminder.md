# Invoke-MgBetaDismissUserEventReminder

## SYNOPSIS
Dismiss a reminder that has been triggered for an event in a user calendar.

> [!NOTE]
> To view the v1.0 release of this cmdlet, view [Invoke-MgDismissUserEventReminder](/powershell/module/Microsoft.Graph.Users.Actions/Invoke-MgDismissUserEventReminder?view=graph-powershell-1.0)

## SYNTAX

### Dismiss (Default)
```
Invoke-MgBetaDismissUserEventReminder -EventId <String> -UserId <String> [-ResponseHeadersVariable <String>]
 [-Headers <IDictionary>] [-PassThru] [-ProgressAction <ActionPreference>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

### DismissViaIdentity
```
Invoke-MgBetaDismissUserEventReminder -InputObject <IUsersActionsIdentity> [-ResponseHeadersVariable <String>]
 [-Headers <IDictionary>] [-PassThru] [-ProgressAction <ActionPreference>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRIPTION
Dismiss a reminder that has been triggered for an event in a user calendar.

### EXAMPLE 1
```
Import-Module Microsoft.Graph.Beta.Users.Actions
# A UPN can also be used as -UserId.
Invoke-MgBetaDismissUserEventReminder -UserId $userId -EventId $eventId
```
