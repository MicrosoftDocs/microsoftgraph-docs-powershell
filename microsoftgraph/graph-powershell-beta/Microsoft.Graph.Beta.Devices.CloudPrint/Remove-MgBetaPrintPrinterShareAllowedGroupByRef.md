# Remove-MgBetaPrintPrinterShareAllowedGroupByRef

## SYNOPSIS
Revoke the specified group's access to submit print jobs to the associated printerShare.

## SYNTAX

### Delete (Default)
```
Remove-MgBetaPrintPrinterShareAllowedGroupByRef -GroupId <String> -PrinterShareId <String> [-IfMatch <String>]
 [-Headers <IDictionary>] [-PassThru] [-ProgressAction <ActionPreference>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

### Delete1
```
Remove-MgBetaPrintPrinterShareAllowedGroupByRef -PrinterShareId <String> -Id <String> [-IfMatch <String>]
 [-Headers <IDictionary>] [-PassThru] [-ProgressAction <ActionPreference>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

### DeleteViaIdentity1
```
Remove-MgBetaPrintPrinterShareAllowedGroupByRef -InputObject <IDevicesCloudPrintIdentity> -Id <String>
 [-IfMatch <String>] [-Headers <IDictionary>] [-PassThru] [-ProgressAction <ActionPreference>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

### DeleteViaIdentity
```

 [-Headers <IDictionary>] [-PassThru] [-ProgressAction <ActionPreference>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```
