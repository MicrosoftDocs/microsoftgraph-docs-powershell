# Get-MgBetaPrintPrinterJobDocumentContent

## SYNOPSIS

Download the binary file associated with a printDocument.
Calling this method yields a redirect response with a preauthenticated URL that can be used to download the payload.

> [!NOTE]
> To view the v1.0 release of this cmdlet, view [Get-MgPrintPrinterJobDocumentContent](/powershell/module/Microsoft.Graph.Devices.CloudPrint/Get-MgPrintPrinterJobDocumentContent?view=graph-powershell-1.0)

## SYNTAX

### Get (Default)

```
Get-MgBetaPrintPrinterJobDocumentContent -PrintDocumentId <string> -PrintJobId <string>
 -PrinterId <string> -OutFile <string> [-ResponseHeadersVariable <string>] [-Break]
 [-Headers <IDictionary>] [-HttpPipelineAppend <SendAsyncStep[]>]
 [-HttpPipelinePrepend <SendAsyncStep[]>] [-PassThru] [-Proxy <uri>]
 [-ProxyCredential <pscredential>] [-ProxyUseDefaultCredentials] [<CommonParameters>]
```

### GetViaIdentity

```
Get-MgBetaPrintPrinterJobDocumentContent -InputObject <IDevicesCloudPrintIdentity> -OutFile <string>
 [-ResponseHeadersVariable <string>] [-Break] [-Headers <IDictionary>]
 [-HttpPipelineAppend <SendAsyncStep[]>] [-HttpPipelinePrepend <SendAsyncStep[]>] [-PassThru]
 [-Proxy <uri>] [-ProxyCredential <pscredential>] [-ProxyUseDefaultCredentials] [<CommonParameters>]
```

## DESCRIPTION

Download the binary file associated with a printDocument.
Calling this method yields a redirect response with a preauthenticated URL that can be used to download the payload.

**Permissions**

| Permission type | Permissions (from least to most privileged) |
| --------------- | ------------------------------------------  |
| Delegated (work or school account) | PrintJob.Read, PrintJob.ReadWrite.All, PrintJob.ReadWrite, PrintJob.Read.All,  |
| Delegated (personal Microsoft account) | Not supported |
| Application | PrintJob.Read.All, PrintJob.ReadWrite.All,  |
