# Set-MgBetaPrintShareJobDocumentContent

## SYNOPSIS
Update media content for the navigation property documents in print

> [!NOTE]
> To view the v1.0 release of this cmdlet, view [Set-MgPrintShareJobDocumentContent](/powershell/module/Microsoft.Graph.Devices.CloudPrint/Set-MgPrintShareJobDocumentContent?view=graph-powershell-v1.0)

## SYNTAX

### Set (Default)
```
Set-MgBetaPrintShareJobDocumentContent -PrintDocumentId <String> -PrintJobId <String> -PrinterShareId <String>
 [-Data <Stream>] -InFile <String> [-PassThru] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### SetViaIdentity
```
Set-MgBetaPrintShareJobDocumentContent -InputObject <IDevicesCloudPrintIdentity> [-Data <Stream>]
 -InFile <String> [-PassThru] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Update media content for the navigation property documents in print
