# Remove-MgBetaSiteOnenotePage

## SYNOPSIS
Delete a OneNote page.

> [!NOTE]
> To view the v1.0 release of this cmdlet, view [Remove-MgSiteOnenotePage](/powershell/module/Microsoft.Graph.Notes/Remove-MgSiteOnenotePage?view=graph-powershell-v1.0)

## SYNTAX

### Delete (Default)
```
Remove-MgBetaSiteOnenotePage -OnenotePageId <String> -SiteId <String> [-IfMatch <String>] [-PassThru] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

### DeleteViaIdentity
```
Remove-MgBetaSiteOnenotePage -InputObject <INotesIdentity> [-IfMatch <String>] [-PassThru] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRIPTION
Delete a OneNote page.
