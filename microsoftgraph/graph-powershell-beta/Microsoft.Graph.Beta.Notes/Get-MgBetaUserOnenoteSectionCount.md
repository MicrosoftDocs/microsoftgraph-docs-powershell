# Get-MgBetaUserOnenoteSectionCount

```
Get-MgBetaUserOnenoteSectionCount -UserId <string> [-Filter <string>] [-Search <string>]
 [-ResponseHeadersVariable <string>] [-Break] [-Headers <IDictionary>]
```
