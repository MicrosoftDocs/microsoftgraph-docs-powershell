# Set-MgUserOnlineMeetingAlternativeRecording

## SYNOPSIS
Update media content for the navigation property onlineMeetings in users

## SYNTAX

### Set (Default)
```
Set-MgUserOnlineMeetingAlternativeRecording -OnlineMeetingId <String> -UserId <String>
 [-BodyParameter <Stream>] -InFile <String> [-PassThru] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### SetViaIdentity
```
Set-MgUserOnlineMeetingAlternativeRecording -InputObject <ICloudCommunicationsIdentity>
 [-BodyParameter <Stream>] -InFile <String> [-PassThru] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Update media content for the navigation property onlineMeetings in users
