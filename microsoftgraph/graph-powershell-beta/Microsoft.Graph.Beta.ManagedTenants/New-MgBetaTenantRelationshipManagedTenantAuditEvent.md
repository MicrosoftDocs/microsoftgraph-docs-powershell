# New-MgBetaTenantRelationshipManagedTenantAuditEvent

## SYNOPSIS

## SYNTAX

### CreateExpanded (Default)

```
New-MgBetaTenantRelationshipManagedTenantAuditEvent [-ResponseHeadersVariable <string>]
 [-Activity <string>] [-ActivityDateTime <datetime>] [-ActivityId <string>]
 [-AdditionalProperties <hashtable>] [-Category <string>] [-HttpVerb <string>] [-IPAddress <string>]
 [-Id <string>] [-InitiatedByAppId <string>] [-InitiatedByUpn <string>]
 [-InitiatedByUserId <string>] [-RequestBody <string>] [-RequestUrl <string>] [-TenantIds <string>]
 [-TenantNames <string>] [-Break] [-Headers <IDictionary>] [-HttpPipelineAppend <SendAsyncStep[]>]
 [-HttpPipelinePrepend <SendAsyncStep[]>] [-Proxy <uri>] [-ProxyCredential <pscredential>]
```
