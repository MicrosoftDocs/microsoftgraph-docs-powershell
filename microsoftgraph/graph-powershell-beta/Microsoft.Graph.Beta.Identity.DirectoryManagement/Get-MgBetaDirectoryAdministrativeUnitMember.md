# Get-MgBetaDirectoryAdministrativeUnitMember

## SYNOPSIS

Users and groups that are members of this administrative unit.
Supports $expand.

## SYNTAX

### List (Default)

```
Get-MgBetaDirectoryAdministrativeUnitMember -AdministrativeUnitId <string>
 [-ExpandProperty <string[]>] [-Filter <string>] [-Property <string[]>] [-Search <string>]
 [-Skip <int>] [-Sort <string[]>] [-Top <int>] [-ConsistencyLevel <string>]
 [-ResponseHeadersVariable <string>] [-Break] [-Headers <IDictionary>]
 [-HttpPipelineAppend <SendAsyncStep[]>] [-HttpPipelinePrepend <SendAsyncStep[]>] [-Proxy <uri>]
 [-ProxyCredential <pscredential>] [-ProxyUseDefaultCredentials] [-PageSize <int>] [-All]
 [-CountVariable <string>] [<CommonParameters>]
```
