# Invoke-MgBetaUnarchiveGroupPlannerPlan

## SYNOPSIS

Unarchive a plannerPlan object.
Unarchiving a plan, also unarchives the plannerTasks and plannerBuckets in the plan.
Only a plan that is archived can be unarchived.
