# Update-MgBetaSearchQna

## SYNOPSIS

> [!NOTE]
> To view the v1.0 release of this cmdlet, view [Update-MgSearchQna](/powershell/module/Microsoft.Graph.Search/Update-MgSearchQna?view=graph-powershell-1.0)
