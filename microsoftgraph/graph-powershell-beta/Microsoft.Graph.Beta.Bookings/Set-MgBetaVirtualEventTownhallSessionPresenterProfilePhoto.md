# Set-MgBetaVirtualEventTownhallSessionPresenterProfilePhoto

## SYNOPSIS
Update profilePhoto for the navigation property presenters in solutions

## SYNTAX

### Set (Default)
```
Set-MgBetaVirtualEventTownhallSessionPresenterProfilePhoto -VirtualEventPresenterId <String>
 -VirtualEventSessionId <String> -VirtualEventTownhallId <String> [-Data <Stream>] -InFile <String> [-PassThru]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

### SetViaIdentity
```
Set-MgBetaVirtualEventTownhallSessionPresenterProfilePhoto -InputObject <IBookingsIdentity> [-Data <Stream>]
 -InFile <String> [-PassThru] [-WhatIf] [-Confirm] [<CommonParameters>]
```
