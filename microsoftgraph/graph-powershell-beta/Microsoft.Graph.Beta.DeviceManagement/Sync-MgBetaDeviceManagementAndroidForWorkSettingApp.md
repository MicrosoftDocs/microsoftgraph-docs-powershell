## SYNOPSIS

## SYNTAX

```
Sync-MgBetaDeviceManagementAndroidForWorkSettingApp [-ResponseHeadersVariable <String>]
 [-Headers <IDictionary>] [-PassThru] [-WhatIf] [-Confirm]
```
