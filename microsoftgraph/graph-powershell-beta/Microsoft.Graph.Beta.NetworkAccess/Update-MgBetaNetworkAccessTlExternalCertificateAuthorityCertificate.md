# Update-MgBetaNetworkAccessTlExternalCertificateAuthorityCertificate

## SYNOPSIS

Update the properties of an externalCertificateAuthorityCertificate object.
This can be used by a GSA admin to upload the generated certificate for GSA.
The certificate is generated by signing the downloaded CSR with the customer's PKI.
