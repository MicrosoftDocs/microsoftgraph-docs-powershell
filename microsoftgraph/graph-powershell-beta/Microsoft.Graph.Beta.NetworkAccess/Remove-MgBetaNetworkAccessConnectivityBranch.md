# Remove-MgBetaNetworkAccessConnectivityBranch

## SYNOPSIS
Delete a specific branch.

## SYNTAX

### Delete (Default)
```
Remove-MgBetaNetworkAccessConnectivityBranch -BranchSiteId <String> [-IfMatch <String>]
 [-ResponseHeadersVariable <String>] [-Headers <IDictionary>] [-PassThru] [-ProgressAction <ActionPreference>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

### DeleteViaIdentity
```
Remove-MgBetaNetworkAccessConnectivityBranch -InputObject <INetworkAccessIdentity> [-IfMatch <String>]
 [-ResponseHeadersVariable <String>] [-Headers <IDictionary>] [-PassThru] [-ProgressAction <ActionPreference>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Delete a specific branch.

**Permissions**

| Permission type | Permissions (from least to most privileged) |
| --------------- | ------------------------------------------  |
| Delegated (work or school account) | NetworkAccess.ReadWrite.All,  |
| Delegated (personal Microsoft account) | Not supported |
| Application | NetworkAccess.ReadWrite.All,  |

## EXAMPLES

### EXAMPLE 1
```
Import-Module Microsoft.Graph.Beta.NetworkAccess
```

Remove-MgBetaNetworkAccessConnectivityBranch -BranchSiteId $branchSiteId
