# Set-MgBetaUserPhotoContent

## SYNOPSIS
The user's profile photo.
Read-only.

> [!NOTE]
> To view the v1.0 release of this cmdlet, view [Set-MgUserPhotoContent](/powershell/module/Microsoft.Graph.Users/Set-MgUserPhotoContent?view=graph-powershell-1.0)

## SYNTAX

### Set (Default)
```
Set-MgBetaUserPhotoContent -UserId <String> [-Data <Stream>] [-ResponseHeadersVariable <String>]
 -InFile <String> [-ContentType <String>] [-Headers <IDictionary>] [-PassThru]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

### Set1
```
Set-MgBetaUserPhotoContent -UserId <String> -ProfilePhotoId <String> [-Data <Stream>]
 [-ResponseHeadersVariable <String>] -InFile <String> [-ContentType <String>] [-Headers <IDictionary>]
 [-PassThru] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### SetViaIdentity1
```
Set-MgBetaUserPhotoContent -InputObject <IUsersIdentity> [-Data <Stream>] [-ResponseHeadersVariable <String>]
 -InFile <String> [-ContentType <String>] [-Headers <IDictionary>] [-PassThru]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

### SetViaIdentity
```
Set-MgBetaUserPhotoContent -InputObject <IUsersIdentity> [-Data <Stream>] [-ResponseHeadersVariable <String>]
 -InFile <String> [-ContentType <String>] [-Headers <IDictionary>] [-PassThru]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
The user's profile photo.
Read-only.

**Permissions**

| Permission type | Permissions (from least to most privileged) |
| --------------- | ------------------------------------------  |
| Delegated (work or school account) | User.ReadWrite, ProfilePhoto.ReadWrite.All,  |
| Delegated (personal Microsoft account) | Not supported |
| Application | ProfilePhoto.ReadWrite.All, User.ReadWrite.All,  |
