# Remove-MgBetaUserTodoListTaskAttachment

## SYNOPSIS
Delete a taskFileAttachment object from a todoTask resource.

> [!NOTE]
> To view the v1.0 release of this cmdlet, view [Remove-MgUserTodoListTaskAttachment](/powershell/module/Microsoft.Graph.Users/Remove-MgUserTodoListTaskAttachment?view=graph-powershell-1.0)

## SYNTAX

### Delete (Default)
```
Remove-MgBetaUserTodoListTaskAttachment -AttachmentBaseId <String> -TodoTaskId <String>
 -TodoTaskListId <String> -UserId <String> [-IfMatch <String>] [-ResponseHeadersVariable <String>]
 [-Headers <IDictionary>] [-PassThru] [-ProgressAction <ActionPreference>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

### DeleteViaIdentity
```
Remove-MgBetaUserTodoListTaskAttachment -InputObject <IUsersIdentity> [-IfMatch <String>]
 [-ResponseHeadersVariable <String>] [-Headers <IDictionary>] [-PassThru] [-ProgressAction <ActionPreference>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Delete a taskFileAttachment object from a todoTask resource.

Import-Module Microsoft.Graph.Beta.Users
