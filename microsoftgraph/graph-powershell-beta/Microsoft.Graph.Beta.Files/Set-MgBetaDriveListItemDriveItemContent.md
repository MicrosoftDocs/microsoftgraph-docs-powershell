# Set-MgBetaDriveListItemDriveItemContent

## SYNOPSIS
The content stream, if the item represents a file.

> [!NOTE]
> To view the v1.0 release of this cmdlet, view [Set-MgDriveListItemDriveItemContent](/powershell/module/Microsoft.Graph.Files/Set-MgDriveListItemDriveItemContent?view=graph-powershell-1.0)

## SYNTAX

### Set (Default)
```
Set-MgBetaDriveListItemDriveItemContent -DriveId <String> -ListItemId <String> [-Data <Stream>]
 [-ResponseHeadersVariable <String>] -InFile <String> [-ContentType <String>] [-Headers <IDictionary>]
 [-ProgressAction <ActionPreference>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### SetViaIdentity
```
Set-MgBetaDriveListItemDriveItemContent -InputObject <IFilesIdentity> [-Data <Stream>]
 [-ResponseHeadersVariable <String>] -InFile <String> [-ContentType <String>] [-Headers <IDictionary>]
 [-ProgressAction <ActionPreference>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
The content stream, if the item represents a file.
