# Update-MgBetaDirectoryObject

## SYNOPSIS

Update entity in directoryObjects

> [!NOTE]
> To view the v1.0 release of this cmdlet, view [Update-MgDirectoryObject](/powershell/module/Microsoft.Graph.DirectoryObjects/Update-MgDirectoryObject?view=graph-powershell-1.0)

## SYNTAX

### UpdateExpanded (Default)

```
Update-MgBetaDirectoryObject -DirectoryObjectId <string> [-ResponseHeadersVariable <string>]
 [-AdditionalProperties <hashtable>] [-DeletedDateTime <datetime>] [-Id <string>] [-Break]
 [-Headers <IDictionary>] [-HttpPipelineAppend <SendAsyncStep[]>]
 [-HttpPipelinePrepend <SendAsyncStep[]>] [-Proxy <uri>] [-ProxyCredential <pscredential>]
 [-ProxyUseDefaultCredentials] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### Update

```
Update-MgBetaDirectoryObject -DirectoryObjectId <string>
 -BodyParameter <IMicrosoftGraphDirectoryObject> [-ResponseHeadersVariable <string>] [-Break]
 [-Headers <IDictionary>] [-HttpPipelineAppend <SendAsyncStep[]>]
 [-HttpPipelinePrepend <SendAsyncStep[]>] [-Proxy <uri>] [-ProxyCredential <pscredential>]
 [-ProxyUseDefaultCredentials] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### UpdateViaIdentityExpanded

```

 [-ResponseHeadersVariable <string>] [-AdditionalProperties <hashtable>]
 [-DeletedDateTime <datetime>] [-Id <string>] [-Break] [-Headers <IDictionary>]
```
