# Get-MgBetaEducationSchoolClassByRef

## SYNOPSIS
Retrieve a list of classes owned by a school.

> [!NOTE]
> To view the v1.0 release of this cmdlet, view [Get-MgEducationSchoolClassByRef](/powershell/module/Microsoft.Graph.Education/Get-MgEducationSchoolClassByRef?view=graph-powershell-v1.0)

## SYNTAX

```
Get-MgBetaEducationSchoolClassByRef -EducationSchoolId <String> [-Filter <String>] [-Search <String>]
 [-Skip <Int32>] [-Sort <String[]>] [-Top <Int32>] [-PageSize <Int32>] [-All] [-CountVariable <String>]
 [<CommonParameters>]
```

## DESCRIPTION
Retrieve a list of classes owned by a school.

## EXAMPLES
### Example 1: Code snippet

```powershell
Import-Module Microsoft.Graph.Beta.Education

Get-MgBetaEducationSchoolClass -EducationSchoolId $educationSchoolId
```
This example shows how to use the Get-MgBetaEducationSchoolClassByRef Cmdlet.

To learn about permissions for this resource, see the [permissions reference](/graph/permissions-reference).
