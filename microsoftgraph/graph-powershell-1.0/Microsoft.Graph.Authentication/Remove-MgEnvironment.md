# Remove-MgEnvironment

## SYNOPSIS

Removes Microsoft Graph environment to the settings file

## SYNTAX

### __AllParameterSets

```
Remove-MgEnvironment [-Name] <string> [-WhatIf] [-Confirm] [<CommonParameters>]
```
