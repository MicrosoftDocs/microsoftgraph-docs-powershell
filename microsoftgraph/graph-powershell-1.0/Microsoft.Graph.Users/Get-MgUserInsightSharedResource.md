# Get-MgUserInsightSharedResource

## SYNOPSIS
Used for navigating to the item that was shared.
For file attachments, the type is fileAttachment.
For linked attachments, the type is driveItem.

## SYNTAX

### Get (Default)
```
Get-MgUserInsightSharedResource -SharedInsightId <String> -UserId <String> [-ExpandProperty <String[]>]
 [-Property <String[]>] [-ResponseHeadersVariable <String>] [-Headers <IDictionary>]
 [-ProgressAction <ActionPreference>] [<CommonParameters>]
```

### GetViaIdentity
```
Get-MgUserInsightSharedResource -InputObject <IUsersIdentity> [-ExpandProperty <String[]>]
 [-Property <String[]>] [-ResponseHeadersVariable <String>] [-Headers <IDictionary>]
 [-ProgressAction <ActionPreference>] [<CommonParameters>]
```

## DESCRIPTION
Used for navigating to the item that was shared.
For file attachments, the type is fileAttachment.
For linked attachments, the type is driveItem.
