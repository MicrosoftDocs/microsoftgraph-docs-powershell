# Invoke-MgRetireDeviceManagementManagedDevice

## SYNOPSIS
Retire a device

## SYNTAX

### Retire (Default)
```
Invoke-MgRetireDeviceManagementManagedDevice -ManagedDeviceId <String> [-ResponseHeadersVariable <String>]
 [-Headers <IDictionary>] [-PassThru] [-ProgressAction <ActionPreference>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

### RetireViaIdentity
```
Invoke-MgRetireDeviceManagementManagedDevice -InputObject <IDeviceManagementIdentity>
 [-ResponseHeadersVariable <String>] [-Headers <IDictionary>] [-PassThru] [-ProgressAction <ActionPreference>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Retire a device
