## SYNTAX

```

 [-AndroidEnrollmentAssignments <IMicrosoftGraphComplianceManagementPartnerAssignment[]>] [-AndroidOnboarded]
 [-DisplayName <String>] [-Id <String>]

 [-LastHeartbeatDateTime <DateTime>]
```
