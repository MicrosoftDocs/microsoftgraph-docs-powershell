# Remove-MgDeviceManagementVirtualEndpointProvisioningPolicy

## SYNOPSIS
Delete a cloudPcProvisioningPolicy object.
You can't delete a policy that's in use.

## SYNTAX

### Delete (Default)
```
Remove-MgDeviceManagementVirtualEndpointProvisioningPolicy -CloudPcProvisioningPolicyId <String>
 [-IfMatch <String>] [-ResponseHeadersVariable <String>] [-Headers <IDictionary>] [-PassThru]
 [-ProgressAction <ActionPreference>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### DeleteViaIdentity
```
Remove-MgDeviceManagementVirtualEndpointProvisioningPolicy
 -InputObject <IDeviceManagementAdministrationIdentity> [-IfMatch <String>] [-ResponseHeadersVariable <String>]
 [-Headers <IDictionary>] [-PassThru] [-ProgressAction <ActionPreference>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRIPTION
Delete a cloudPcProvisioningPolicy object.
You can't delete a policy that's in use.
