# Stop-MgPrintShareJob

## SYNOPSIS
Cancel a print job.
Print jobs can be canceled only on behalf of a user, using delegated permissions.

> [!NOTE]
> To view the beta release of this cmdlet, view [Stop-MgBetaPrintShareJob](/powershell/module/Microsoft.Graph.Beta.Devices.CloudPrint/Stop-MgBetaPrintShareJob?view=graph-powershell-beta)

## SYNTAX

### Cancel (Default)
```
Stop-MgPrintShareJob -PrintJobId <String> -PrinterShareId <String> [-ResponseHeadersVariable <String>]
 [-Headers <IDictionary>] [-PassThru] [-ProgressAction <ActionPreference>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

### CancelViaIdentity
```
Stop-MgPrintShareJob -InputObject <IDevicesCloudPrintIdentity> [-ResponseHeadersVariable <String>]
 [-Headers <IDictionary>] [-PassThru] [-ProgressAction <ActionPreference>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRIPTION
Cancel a print job.
Print jobs can be canceled only on behalf of a user, using delegated permissions.
