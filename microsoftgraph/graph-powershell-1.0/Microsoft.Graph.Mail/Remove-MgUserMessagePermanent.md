# Remove-MgUserMessagePermanent

## SYNOPSIS
Invoke action permanentDelete

> [!NOTE]
> To view the beta release of this cmdlet, view [Remove-MgBetaUserMessagePermanent](/powershell/module/Microsoft.Graph.Beta.Mail/Remove-MgBetaUserMessagePermanent?view=graph-powershell-beta)

## SYNTAX

### Delete (Default)
```
Remove-MgUserMessagePermanent -MessageId <String> -UserId <String> [-ResponseHeadersVariable <String>]
 [-Headers <IDictionary>] [-PassThru] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

### DeleteViaIdentity
```
Remove-MgUserMessagePermanent -InputObject <IMailIdentity> [-ResponseHeadersVariable <String>]
 [-Headers <IDictionary>] [-PassThru] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRIPTION
Invoke action permanentDelete

**Permissions**

| Permission type | Permissions (from least to most privileged) |
| --------------- | ------------------------------------------  |
| Delegated (work or school account) | Mail.ReadWrite,  |
| Delegated (personal Microsoft account) | Mail.ReadWrite,  |
| Application | Mail.ReadWrite,  |
