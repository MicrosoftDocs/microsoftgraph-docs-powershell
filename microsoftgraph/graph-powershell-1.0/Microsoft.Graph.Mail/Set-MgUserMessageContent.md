# Set-MgUserMessageContent

## SYNOPSIS
Update media content for the navigation property messages in users

> [!NOTE]
> To view the beta release of this cmdlet, view [Set-MgBetaUserMessageContent](/powershell/module/Microsoft.Graph.Beta.Mail/Set-MgBetaUserMessageContent?view=graph-powershell-beta)

## SYNTAX

### Set (Default)
```
Set-MgUserMessageContent -MessageId <String> -UserId <String> [-Data <Stream>] -InFile <String> [-PassThru]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

### SetViaIdentity
```
Set-MgUserMessageContent -InputObject <IMailIdentity> [-Data <Stream>] -InFile <String> [-PassThru] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Update media content for the navigation property messages in users
