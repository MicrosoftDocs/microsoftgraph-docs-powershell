# Set-MgUserChatMessageReplyHostedContent

## SYNOPSIS
Update media content for the navigation property hostedContents in users

> [!NOTE]
> To view the beta release of this cmdlet, view [Set-MgBetaUserChatMessageReplyHostedContent](/powershell/module/Microsoft.Graph.Beta.Teams/Set-MgBetaUserChatMessageReplyHostedContent?view=graph-powershell-beta)

## SYNTAX

### Set (Default)
```
Set-MgUserChatMessageReplyHostedContent -ChatId <String> -ChatMessageHostedContentId <String>
 -ChatMessageId <String> -ChatMessageId1 <String> -UserId <String> [-Data <Stream>] -InFile <String>
 [-PassThru] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### SetViaIdentity
```
Set-MgUserChatMessageReplyHostedContent -InputObject <ITeamsIdentity> [-Data <Stream>] -InFile <String>
 [-PassThru] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Update media content for the navigation property hostedContents in users
