# New-MgGroupLifecyclePolicy

## SYNOPSIS

Creates a new groupLifecyclePolicy.
Only one policy exists in the tenant.

## DESCRIPTION

Creates a new groupLifecyclePolicy.

**Permissions**

| Permission type | Permissions (from least to most privileged) |
