# New-MgSolutionBackupRestoreSiteProtectionUnit

## SYNOPSIS

Create new navigation property to siteProtectionUnits for solutions

## SYNTAX

### CreateExpanded (Default)

```
New-MgSolutionBackupRestoreSiteProtectionUnit [-ResponseHeadersVariable <string>]
 [-AdditionalProperties <hashtable>] [-CreatedBy <IMicrosoftGraphIdentitySet>]
 [-CreatedDateTime <datetime>] [-Error <IMicrosoftGraphPublicError>] [-Id <string>]
 [-LastModifiedBy <IMicrosoftGraphIdentitySet>] [-LastModifiedDateTime <datetime>]
 [-PolicyId <string>] [-SiteId <string>] [-Status <string>] [-Break] [-Headers <IDictionary>]
 [-HttpPipelineAppend <SendAsyncStep[]>] [-HttpPipelinePrepend <SendAsyncStep[]>] [-Proxy <uri>]
 [-ProxyCredential <pscredential>] [-ProxyUseDefaultCredentials] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```
