# New-MgSolutionBackupRestoreSiteProtectionUnit

## SYNOPSIS
Create new navigation property to siteProtectionUnits for solutions

> [!NOTE]
> To view the beta release of this cmdlet, view [New-MgBetaSolutionBackupRestoreSiteProtectionUnit](/powershell/module/Microsoft.Graph.Beta.BackupRestore/New-MgBetaSolutionBackupRestoreSiteProtectionUnit?view=graph-powershell-beta)

## SYNTAX

### CreateExpanded (Default)
```
New-MgSolutionBackupRestoreSiteProtectionUnit [-ResponseHeadersVariable <String>]
 [-AdditionalProperties <Hashtable>] [-CreatedBy <IMicrosoftGraphIdentitySet>] [-CreatedDateTime <DateTime>]
 [-Error <IMicrosoftGraphPublicError>] [-Id <String>] [-LastModifiedBy <IMicrosoftGraphIdentitySet>]
 [-LastModifiedDateTime <DateTime>] [-PolicyId <String>] [-SiteId <String>] [-Status <String>]
 [-Headers <IDictionary>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### Create
```
New-MgSolutionBackupRestoreSiteProtectionUnit -BodyParameter <IMicrosoftGraphSiteProtectionUnit>
 [-ResponseHeadersVariable <String>] [-Headers <IDictionary>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
