# Get-MgGroupSiteGetByPathPermission

## SYNOPSIS
Get the permission resources from the permissions navigation property on a site.

> [!NOTE]
> To view the beta release of this cmdlet, view [Get-MgBetaGroupSiteGetByPathPermission](/powershell/module/Microsoft.Graph.Beta.Sites/Get-MgBetaGroupSiteGetByPathPermission?view=graph-powershell-beta)

## SYNTAX

### List1 (Default)
```
Get-MgGroupSiteGetByPathPermission -GroupId <String> -Path <String> -SiteId <String>
 [-ExpandProperty <String[]>] [-Filter <String>] [-Property <String[]>] [-Search <String>] [-Skip <Int32>]
 [-Sort <String[]>] [-Top <Int32>] [-Headers <IDictionary>] [-PageSize <Int32>] [-All]
 [-CountVariable <String>] [-ProgressAction <ActionPreference>] [<CommonParameters>]
```

### List
```
Get-MgGroupSiteGetByPathPermission -GroupId <String> -Path <String> -SiteId <String> -Path1 <String>
 [-ExpandProperty <String[]>] [-Filter <String>] [-Property <String[]>] [-Search <String>] [-Skip <Int32>]
 [-Sort <String[]>] [-Top <Int32>] [-Headers <IDictionary>] [-PageSize <Int32>] [-All]
 [-CountVariable <String>] [-ProgressAction <ActionPreference>] [<CommonParameters>]
```

## DESCRIPTION
Get the permission resources from the permissions navigation property on a site.
