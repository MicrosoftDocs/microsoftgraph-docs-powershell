# Update-MgSiteListItemDocumentSetVersion

## SYNOPSIS

Update the navigation property documentSetVersions in sites

> [!NOTE]
> To view the beta release of this cmdlet, view [Update-MgBetaSiteListItemDocumentSetVersion](/powershell/module/Microsoft.Graph.Beta.Sites/Update-MgBetaSiteListItemDocumentSetVersion?view=graph-powershell-beta)

### UpdateExpanded (Default)

```
Update-MgSiteListItemDocumentSetVersion -DocumentSetVersionId <string> -ListId <string>
 -ListItemId <string> -SiteId <string> [-ResponseHeadersVariable <string>]
 [-AdditionalProperties <hashtable>] [-Comment <string>] [-CreatedBy <IMicrosoftGraphIdentitySet>]

 [-Items <IMicrosoftGraphDocumentSetVersionItem[]>] [-LastModifiedBy <IMicrosoftGraphIdentitySet>]

 [<CommonParameters>]
```
