# Remove-MgPrivacySubjectRightsRequest

## SYNOPSIS

### Delete (Default)
```
Remove-MgPrivacySubjectRightsRequest -SubjectRightsRequestId <String> [-IfMatch <String>]
 [-ResponseHeadersVariable <String>] [-Headers <IDictionary>] [-PassThru] [-ProgressAction <ActionPreference>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```
