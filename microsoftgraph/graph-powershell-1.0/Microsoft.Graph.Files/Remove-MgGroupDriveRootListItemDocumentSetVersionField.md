# Remove-MgGroupDriveRootListItemDocumentSetVersionField

## SYNOPSIS

Delete navigation property fields for groups

> [!NOTE]
> To view the beta release of this cmdlet, view [Remove-MgBetaGroupDriveRootListItemDocumentSetVersionField](/powershell/module/Microsoft.Graph.Beta.Files/Remove-MgBetaGroupDriveRootListItemDocumentSetVersionField?view=graph-powershell-beta)

## SYNTAX

### Delete (Default)

```
Remove-MgGroupDriveRootListItemDocumentSetVersionField -DocumentSetVersionId <string>
 -DriveId <string> -GroupId <string> [-IfMatch <string>] [-ResponseHeadersVariable <string>]
 [-Break] [-Headers <IDictionary>] [-HttpPipelineAppend <SendAsyncStep[]>]
 [-HttpPipelinePrepend <SendAsyncStep[]>] [-PassThru] [-Proxy <uri>]
 [-ProxyCredential <pscredential>] [-ProxyUseDefaultCredentials] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

### DeleteViaIdentity

```
Remove-MgGroupDriveRootListItemDocumentSetVersionField -InputObject <IFilesIdentity>
 [-IfMatch <string>] [-ResponseHeadersVariable <string>] [-Break] [-Headers <IDictionary>]
 [-HttpPipelineAppend <SendAsyncStep[]>] [-HttpPipelinePrepend <SendAsyncStep[]>] [-PassThru]
 [-Proxy <uri>] [-ProxyCredential <pscredential>] [-ProxyUseDefaultCredentials] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```
