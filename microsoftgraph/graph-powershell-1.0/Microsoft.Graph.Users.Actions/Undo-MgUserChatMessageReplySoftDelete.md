# Undo-MgUserChatMessageReplySoftDelete

## SYNOPSIS
Undo soft deletion of a single chatMessage or a chat message reply in a channel or a chat.

## SYNTAX

### Undo (Default)
```
Undo-MgUserChatMessageReplySoftDelete -ChatId <String> -ChatMessageId <String> -ChatMessageId1 <String>
 -UserId <String> [-ResponseHeadersVariable <String>] [-Headers <IDictionary>] [-PassThru]
 [-ProgressAction <ActionPreference>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### UndoViaIdentity
```
Undo-MgUserChatMessageReplySoftDelete -InputObject <IUsersActionsIdentity> [-ResponseHeadersVariable <String>]
 [-Headers <IDictionary>] [-PassThru] [-ProgressAction <ActionPreference>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```
