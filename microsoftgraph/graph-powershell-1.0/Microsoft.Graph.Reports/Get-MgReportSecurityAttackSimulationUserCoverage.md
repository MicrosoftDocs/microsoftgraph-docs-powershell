# Get-MgReportSecurityAttackSimulationUserCoverage

## SYNOPSIS

List training coverage for each tenant user in attack simulation and training campaigns.
This function supports @odata.nextLink for pagination.

## SYNTAX

### Get (Default)

```
Get-MgReportSecurityAttackSimulationUserCoverage [-Count] [-Filter <string>] [-Search <string>]
 [-Skip <int>] [-Top <int>] [-ResponseHeadersVariable <string>] [-Break] [-Headers <IDictionary>]
 [-HttpPipelineAppend <SendAsyncStep[]>] [-HttpPipelinePrepend <SendAsyncStep[]>] [-Proxy <uri>]
 [-ProxyCredential <pscredential>] [-ProxyUseDefaultCredentials] [<CommonParameters>]
```
