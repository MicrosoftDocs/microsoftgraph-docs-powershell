# Remove-MgPlannerPlan

## SYNOPSIS
Delete a plannerPlan object.

## SYNTAX

### Delete (Default)
```
Remove-MgPlannerPlan -PlannerPlanId <String> [-IfMatch <String>] [-ResponseHeadersVariable <String>]
 [-Headers <IDictionary>] [-PassThru] [-ProgressAction <ActionPreference>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

### DeleteViaIdentity
```
Remove-MgPlannerPlan -InputObject <IPlannerIdentity> [-IfMatch <String>] [-ResponseHeadersVariable <String>]
 [-Headers <IDictionary>] [-PassThru] [-ProgressAction <ActionPreference>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRIPTION
Delete a plannerPlan object.
